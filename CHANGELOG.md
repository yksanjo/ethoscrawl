# Changelog

All notable changes to EthosCrawl will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-20

### Added
- Initial release of EthosCrawl
- Core ethical scraping framework
- Privacy engine with 4 privacy levels (minimal, standard, strict, maximum)
- Ethical framework with GDPR and CCPA compliance
- Adaptive scraping engine with fallback support
- Basic CLI interface
- Example scripts and documentation
- Rate limiting and session management
- Transparency reporting infrastructure

### Features
- 🔒 Privacy preservation with multiple levels
- ⚖️ Ethical compliance checking
- 🤖 Adaptive scraping that learns from website changes
- 📊 Detailed metadata and transparency reports
- 🚫 Built-in rate limiting
- 🔍 Support for CSS selectors with pseudo-elements
- 🐍 Async-first Python API

### Technical Details
- Python 3.8+ compatibility
- Async/await architecture
- Type hints throughout
- Modular design for easy extension
- Comprehensive test suite
