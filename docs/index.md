## 📖 About This Documentation

This documentation is built with [MkDocs](https://www.mkdocs.org/) and deployed to GitHub Pages. You can contribute to the documentation by editing the Markdown files in the `docs/` directory.

### Building Locally

```bash
# Install mkdocs
pip install mkdocs mkdocs-material

# Serve documentation locally
mkdocs serve

# Build static site
mkdocs build
```
